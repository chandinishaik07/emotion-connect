##  Emotion Connect

Emotion Connect is an anonymous emotion-based chat platform that allows users to connect with others who feel the same  without identity pressure.

---

##  Live Demo

 https://emotion-connect.onrender.com  

---

##  Problem Statement

Many people feel stressed, anxious, lonely, or overwhelmed but hesitate to express themselves due to:

- Fear of judgment  
- Social pressure  
- Lack of safe spaces  
- No instant emotional support  

There is no simple platform where people can connect based on emotions rather than identity.

---

##  Our Solution

Emotion Connect provides:

- Anonymous usernames  
- Emotion-based chat rooms  
- No login required  
- Instant connection  
- Safe digital interaction  

Users select how they feel and join a chat room with others feeling the same.

##  How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/chandinishaik07/emotion-connect.git
   ```

2. Navigate to the project folder:
   ```
   cd emotion-connect
   ```

3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Run the app:
   ```
   python app.py
   ```

5. Open in browser:
   ```
   http://127.0.0.1:5000
   ```
